Como primer paso ingresa desde el link https://github.com/paulinaguida10-debug/ssl-antlr-calculator a github.
Luego presiona el boton verde (“code”/“codigo”), y copia el URL.
Como tercer paso abri una nueva terminar.
Escribi git clone URL copiado en la terminal,es decir, quedaria algo asi: git clone https://github.com/paulinaguida10-debug/ssl-antlr-calculator.git y apretar enter
Escribi cd ssl-antlr-calculator y presiona enter
Luego escribir npm install y presiona enter.
Y para probar el ejemplo 1 escribir node index.js ejemplo_valido1.
Entonces asi se mostrara el resultado con su respectivo arbol de derivacion.
